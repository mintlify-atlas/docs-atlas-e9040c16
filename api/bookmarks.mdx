The `BookmarksService` handles all bookmark-related operations including creating, retrieving, updating, and deleting bookmarks.

## Accessing the Service

```go
client := linkvite.NewClient("your-api-key")
bookmarks := client.Bookmarks
```

## List Operations

### List

Retrieves bookmarks for the current user with pagination.

```go
func (s *BookmarksService) List(ctx context.Context, opts *ListOptions) (*ListBookmarksResult, error)
```

<ParamField path="ctx" type="context.Context" required>
  Context for the request
</ParamField>

<ParamField path="opts" type="*ListOptions">
  Pagination and filtering options
  
  <Expandable title="ListOptions fields">
    <ParamField path="Page" type="int64">
      Page number (starting from 1)
    </ParamField>
    <ParamField path="Limit" type="int64">
      Number of items per page
    </ParamField>
    <ParamField path="Query" type="string">
      Search query
    </ParamField>
    <ParamField path="Status" type="string">
      Filter by status: "active" or "trashed"
    </ParamField>
    <ParamField path="Sort" type="string">
      Sort order
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="ListBookmarksResult" type="*ListBookmarksResult">
  <Expandable title="fields">
    <ResponseField name="Bookmarks" type="[]*BookmarkWithMeta">
      List of bookmarks with metadata
    </ResponseField>
    <ResponseField name="Pagination" type="*Pagination">
      Pagination information
    </ResponseField>
  </Expandable>
</ResponseField>

**Example**

```go
opts := &linkvite.ListOptions{
    Page:  1,
    Limit: 20,
    Query: "golang",
    Status: "active",
}

result, err := client.Bookmarks.List(context.Background(), opts)
if err != nil {
    log.Fatal(err)
}

for _, bookmark := range result.Bookmarks {
    fmt.Printf("%s: %s\n", bookmark.Title, bookmark.URL)
}
```

## Create Operations

### Create

Creates a new bookmark.

```go
func (s *BookmarksService) Create(ctx context.Context, input *CreateBookmarkInput) (*Bookmark, error)
```

<ParamField path="ctx" type="context.Context" required>
  Context for the request
</ParamField>

<ParamField path="input" type="*CreateBookmarkInput" required>
  <Expandable title="CreateBookmarkInput fields">
    <ParamField path="URL" type="string" required>
      The URL to bookmark
    </ParamField>
    <ParamField path="Collection" type="string">
      Collection ID to add the bookmark to
    </ParamField>
    <ParamField path="Title" type="string">
      Custom title (auto-detected if not provided)
    </ParamField>
    <ParamField path="Description" type="string">
      Custom description
    </ParamField>
    <ParamField path="Starred" type="*bool">
      Whether to star the bookmark
    </ParamField>
    <ParamField path="Tags" type="string">
      Comma-separated tags
    </ParamField>
    <ParamField path="Cover" type="string">
      Custom cover image URL
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="Bookmark" type="*Bookmark">
  The created bookmark
</ResponseField>

**Example**

```go
starred := true
bookmark, err := client.Bookmarks.Create(context.Background(), &linkvite.CreateBookmarkInput{
    URL:         "https://golang.org/doc/",
    Title:       "Go Documentation",
    Description: "Official Go programming language documentation",
    Tags:        "golang,programming,documentation",
    Starred:     &starred,
    Collection:  "my-collection-id",
})
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Created bookmark: %s\n", bookmark.ID)
```

## Other Operations

### Exists

Checks if a bookmark with the given URL exists.

```go
func (s *BookmarksService) Exists(ctx context.Context, input *BookmarkExistsInput) (*BookmarkExistsResult, error)
```

<ParamField path="ctx" type="context.Context" required>
  Context for the request
</ParamField>

<ParamField path="input" type="*BookmarkExistsInput" required>
  <Expandable title="BookmarkExistsInput fields">
    <ParamField path="URL" type="string" required>
      URL to check
    </ParamField>
    <ParamField path="Collection" type="string">
      Optional collection ID to scope the check
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="BookmarkExistsResult" type="*BookmarkExistsResult">
  <Expandable title="fields">
    <ResponseField name="Exists" type="bool">
      Whether the bookmark exists
    </ResponseField>
    <ResponseField name="Bookmark" type="*Bookmark">
      The existing bookmark (if found)
    </ResponseField>
  </Expandable>
</ResponseField>

**Example**

```go
result, err := client.Bookmarks.Exists(context.Background(), &linkvite.BookmarkExistsInput{
    URL:        "https://example.com",
    Collection: "my-collection",
})
if err != nil {
    log.Fatal(err)
}

if result.Exists {
    fmt.Printf("Bookmark exists: %s\n", result.Bookmark.Title)
} else {
    fmt.Println("Bookmark does not exist")
}
```

## Error Handling

All methods return an error as the last return value. Always check for errors:

```go
bookmark, err := client.Bookmarks.Get(context.Background(), "bookmark-123")
if err != nil {
    // Handle error
    log.Printf("Failed to get bookmark: %v", err)
    return
}

// Use bookmark
fmt.Println(bookmark.Title)
```
