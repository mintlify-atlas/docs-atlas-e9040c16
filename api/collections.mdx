The `CollectionsService` handles all collection-related operations including creating, managing, and collaborating on collections.

## Accessing the Service

```go
client := linkvite.NewClient("your-api-key")
collections := client.Collections
```

## List and Retrieve Operations

### List

Retrieves all collections for the current user.

```go
func (s *CollectionsService) List(ctx context.Context, opts *ListOptions) ([]*CollectionWithMeta, error)
```

<ParamField path="ctx" type="context.Context" required>
  Context for the request
</ParamField>

<ParamField path="opts" type="*ListOptions">
  Pagination and filtering options
</ParamField>

<ResponseField name="collections" type="[]*CollectionWithMeta">
  Array of collections with metadata including owner and bookmark count
</ResponseField>

**Example**

```go
collections, err := client.Collections.List(context.Background(), &linkvite.ListOptions{
    Limit: 50,
    Sort:  "name",
})
if err != nil {
    log.Fatal(err)
}

for _, col := range collections {
    fmt.Printf("%s (%d bookmarks)\n", col.Name, col.BookmarksCount)
}
```

### Get

Retrieves a single collection by ID.

```go
func (s *CollectionsService) Get(ctx context.Context, id string) (*CollectionWithMeta, error)
```

<ParamField path="ctx" type="context.Context" required>
  Context for the request
</ParamField>

<ParamField path="id" type="string" required>
  Collection ID
</ParamField>

<ResponseField name="collection" type="*CollectionWithMeta">
  The collection with metadata
</ResponseField>

**Example**

```go
collection, err := client.Collections.Get(context.Background(), "collection-123")
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Name: %s\nOwner: %s\nBookmarks: %d\n",
    collection.Name,
    collection.OwnerName,
    collection.BookmarksCount,
)
```

### Create

Creates a new collection.

```go
func (s *CollectionsService) Create(ctx context.Context, input *CreateCollectionInput) (*Collection, error)
```

<ParamField path="ctx" type="context.Context" required>

</ParamField>

<ParamField path="input" type="*CreateCollectionInput" required>

  </Expandable>


## Organization Operations

### Move

Moves a collection to a new parent or removes it from parent.

```go
func (s *CollectionsService) Move(ctx context.Context, id string, input *MoveCollectionInput) error
```

<ParamField path="ctx" type="context.Context" required>
  Context for the request
</ParamField>

<ParamField path="id" type="string" required>
  Collection ID to move
</ParamField>

<ParamField path="input" type="*MoveCollectionInput" required>
  <Expandable title="MoveCollectionInput fields">
    <ParamField path="Parent" type="*string">
      New parent collection ID
    </ParamField>
    <ParamField path="Remove" type="*bool">
      Set to true to remove from parent (move to root)
    </ParamField>
  </Expandable>
</ParamField>

**Example**

```go
// Move to a new parent
parent := "new-parent-id"
err := client.Collections.Move(context.Background(), "collection-123", &linkvite.MoveCollectionInput{
    Parent: &parent,
})

// Move to root (remove parent)
remove := true
err = client.Collections.Move(context.Background(), "collection-123", &linkvite.MoveCollectionInput{
    Remove: &remove,
})
```

## Member Management

### GetMembers

Retrieves the members of a collection.

```go
func (s *CollectionsService) GetMembers(ctx context.Context, id string) (*CollectionMembersResult, error)
```

<ParamField path="ctx" type="context.Context" required>
  Context for the request
</ParamField>

<ParamField path="id" type="string" required>
  Collection ID
</ParamField>

<ResponseField name="result" type="*CollectionMembersResult">
  <Expandable title="fields">
    <ResponseField name="Role" type="RoleLevel">
      Current user's role ("admin" or "viewer")
    </ResponseField>
    <ResponseField name="Members" type="[]*CollectionMember">
      Array of collection members
    </ResponseField>
    <ResponseField name="OwnerID" type="string">
      ID of the collection owner
    </ResponseField>
  </Expandable>
</ResponseField>

**Example**

```go
members, err := client.Collections.GetMembers(context.Background(), "collection-123")
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Your role: %s\n", members.Role)
fmt.Printf("Total members: %d\n", len(members.Members))

for _, member := range members.Members {
    fmt.Printf("- %s (@%s): %s\n", member.Name, member.Username, member.Role)
}
```

## Error Handling

All methods return an error. Always check for errors:

```go
collection, err := client.Collections.Get(context.Background(), "collection-123")
if err != nil {
    log.Printf("Failed to get collection: %v", err)
    return
}

fmt.Println(collection.Name)
```
