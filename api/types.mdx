## User Types

### User

Represents a Linkvite user account with full details including settings and preferences.

```go
type User struct {
    ID             string
    Name           string
    Email          string
    Username       string
    Avatar         string
    EmailVerified  bool
    AccountType    AccountType
    Status         UserStatus
    Verified       bool
    FolderName     string
    PrivateAccount bool
    LastLoginAt    *time.Time
    CreatedAt      time.Time
    UpdatedAt      time.Time
    PushEnabled    bool
    EmailEnabled   bool
    InAppEnabled   bool
}
```

<ResponseField name="ID" type="string">
  Unique identifier for the user
</ResponseField>

<ResponseField name="Name" type="string">

</ResponseField>

  Email address of the user
</ResponseField>

<ResponseField name="Username" type="string">
  Unique username for the user
</ResponseField>

<ResponseField name="Avatar" type="string">
