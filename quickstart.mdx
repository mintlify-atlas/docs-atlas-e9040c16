This guide will walk you through creating your first Linkvite application. You'll learn how to install the SDK, authenticate, and create your first bookmark.

## Prerequisites

Before you begin, make sure you have:

- Go 1.24.5 or later installed
- A Linkvite account (sign up at [linkvite.io](https://linkvite.io))
- An API key from your Linkvite dashboard

<Note>
  New to Linkvite? [Create a free account](https://linkvite.io) to get started.
</Note>

## Step 1: Install the SDK

<Steps>
  <Step title="Install the package">
    Use `go get` to install the Linkvite Go SDK:
    
    ```bash
    go get github.com/linkvite/go
    ```
  </Step>
  
  <Step title="Initialize your project">
    Create a new Go module if you haven't already:
    
    ```bash
    mkdir my-linkvite-app
    cd my-linkvite-app
    go mod init my-linkvite-app
    go get github.com/linkvite/go
    ```
  </Step>
</Steps>

## Step 2: Get your API key

<Steps>
  <Step title="Log in to Linkvite">
    Visit [linkvite.io](https://linkvite.io) and log in to your account.
  </Step>
  
  <Step title="Navigate to settings">
    Go to your account settings and find the API Keys section.
  </Step>
  
  <Step title="Create an API key">
    Click "Create API Key" and give it a name (e.g., "My Go App").
    
    <Warning>
      Copy your API key immediately - you won't be able to see it again! API keys start with `link_`.
    </Warning>
  </Step>
  
  <Step title="Store securely">
    Save your API key in an environment variable:
    
    ```bash
    export LINKVITE_API_KEY="link_your-api-key-here"
    ```
  </Step>
</Steps>

## Step 3: Create your first application

Create a file called `main.go` with the following code:

```go main.go
package main

import (
    "context"
    "fmt"
    "log"
    "os"

    linkvite "github.com/linkvite/go"
)

func main() {
    // Get API key from environment
    apiKey := os.Getenv("LINKVITE_API_KEY")
    if apiKey == "" {
        log.Fatal("LINKVITE_API_KEY environment variable not set")
    }

    // Create a new client
    client, err := linkvite.NewClient(apiKey)
    if err != nil {
        log.Fatal(err)
    }

    ctx := context.Background()

    // Get current user to verify authentication
    user, err := client.User.Get(ctx)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Hello, %s!\n", user.Name)

    // Create your first bookmark
    bookmark, err := client.Bookmarks.Create(ctx, &linkvite.CreateBookmarkInput{
        URL:         "https://go.dev",
        Title:       "Go Programming Language",
        Description: "Official Go website",
    })
    if err != nil {
        log.Fatal(err)
    }

    fmt.Printf("Created bookmark: %s\n", bookmark.Title)
    fmt.Printf("Bookmark ID: %s\n", bookmark.ID)
    fmt.Printf("URL: %s\n", bookmark.URL)
}
```

## Step 4: Run your application

Run your application:

```bash
go run main.go
```

You should see output like:

```
Hello, John Doe!
Created bookmark: Go Programming Language
Bookmark ID: bmk_abc123
URL: https://go.dev
```

<Check>
  Congratulations! You've successfully created your first Linkvite application.
</Check>

## Common operations

Here are some common operations you might want to try next:

### List your bookmarks

```go
// List bookmarks with pagination
result, err := client.Bookmarks.List(ctx, &linkvite.ListOptions{
    Page:  1,
    Limit: 20,
})
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Found %d bookmarks\n", result.Pagination.Total)
for _, bookmark := range result.Bookmarks {
    fmt.Printf("- %s: %s\n", bookmark.Title, bookmark.URL)
}
```

### Create a collection

```go
collection, err := client.Collections.Create(ctx, &linkvite.CreateCollectionInput{
    Name:        "Go Resources",
    Description: "Useful Go programming resources",
    IsPrivate:   linkvite.Ptr(false),
})
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Created collection: %s (ID: %s)\n", collection.Name, collection.ID)
```

### Add a bookmark to a collection

```go
bookmark, err := client.Bookmarks.Create(ctx, &linkvite.CreateBookmarkInput{
    URL:        "https://go.dev/doc/",
    Title:      "Go Documentation",
    Collection: collection.ID,
})
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Added bookmark to collection: %s\n", bookmark.Title)
```

### Search bookmarks

```go
results, err := client.Search.Search(ctx, &linkvite.SearchInput{
    Query: "golang",
    Type:  "bookmarks",
    Page:  1,
    Limit: 10,
})
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Found %d results for 'golang'\n", results.Pagination.Total)
```

## Error handling

Always check for errors and handle them appropriately:

```go
bookmark, err := client.Bookmarks.Get(ctx, "bmk_invalid")
if err != nil {
    if apiErr, ok := err.(*linkvite.Error); ok {
        if apiErr.IsNotFound() {
            fmt.Println("Bookmark not found")
        } else if apiErr.IsUnauthorized() {
            fmt.Println("Invalid API key")
        } else {
            fmt.Printf("API error: %s\n", apiErr.Message)
        }
    } else {
        fmt.Printf("Error: %v\n", err)
    }
}
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Authentication failed">
    Make sure your API key is correct and starts with `link_`. Check that you've set the `LINKVITE_API_KEY` environment variable.
  </Accordion>
  
  <Accordion title="Package not found">
    Run `go mod tidy` to ensure all dependencies are downloaded. Make sure you're using Go 1.24.5 or later.
  </Accordion>
  
  <Accordion title="Connection timeout">
    If you're behind a proxy or firewall, you may need to configure your HTTP client. See the [Client configuration guide](/concepts/client) for details.
  </Accordion>
</AccordionGroup>
